Kamon Online

Kamon is a two-player board game which combines elements of area control (like Go) but in a simple and attractive package. This is an online implementation of Kamon that allows two players to play against each other over the internet. To create a game, type in a room code (this can be anything) and click "Host". This room code can be given to the other player, who types this in and clicks "Join". Please read the rules to get an idea of how the game works.

Rules

The rules of Kamon are simple. White begins by choosing any tile (a "kamon") on the board. Black may respond by choosing any kamon that has either the same colour or the same symbol. There are three winning conditions:

  1. A player connects two opposite sides of the board.
  2. A player surrounds a region containing at least one blank kamon, or at least one kamon of the opposing player.
  3. A player immediately wins if an opponent is unable to make a valid move.
To start a new game, simply click "Host" again.
